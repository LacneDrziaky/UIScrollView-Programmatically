# UISCrollView Programmatically (Swift)

Creating a UIScrollView using Autolayout (programmatically) with auto-height.

## Features

* Programmatically (no storyboard / nib)
* Autolayout
* UIScrollView
* ContentView inside UIScrollView
* Dynamic height
* Handles rotation
